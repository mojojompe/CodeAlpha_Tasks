# Code Alpha Tasks

This repository contains various machine learning and data science projects developed by Emmanuel Ayomiposi, Machine Learning Intern - Code Alpha.

## Projects

### 1. CodeAlpha_Credit_Scoring_Model
A machine learning model designed to predict creditworthiness based on applicant data.
*   **Technologies:** Python, Pandas, NumPy, Scikit-learn
*   **Key Files:** `credit_scoring_model.py`

### 2. CodeAlpha_Disease_Prediction
A disease prediction system utilizing ensemble learning techniques.
*   **Technologies:** Python, XGBoost, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn
*   **Key Features:** Data analysis, model training with XGBoost, and performance visualization.

### 3. CodeAlpha_Handwritten_Character_Recognition
A deep learning project for recognizing handwritten characters from images.
*   **Technologies:** Python, PyTorch, Torchvision, OpenCV, Scikit-learn, Pillow
*   **Architecture:** Convolutional Neural Network (CNN) implemented in `src/model.py`.

## Information.

Each project is contained within its own directory. Navigate to the specific directory to find more details and run the code.
